Dernière mise à jour : 14 mai 2026

2. Quelles données collectons-nous ?

2.1 Données que tu nous fournis volontairement

  • Compte : adresse email, pseudo, mot de passe (hashé via Supabase Auth)
  • Profil : bio, photo de profil, lien social (optionnel)
  • Contenu : articles, clips, commentaires, sources, messages privés
  • Signalements : raison + détails que tu fournis si tu signales un contenu

2.2 Données collectées automatiquement

  • Logs techniques : IP, type d'appareil, OS, version d'app — stockés 30 jours, usage débogage et sécurité
  • Stats anonymisées : lectures par article, durées moyennes — sans lien direct avec ton identité
  • Notifications push : identifiant device APNs/FCM si tu actives les notifs

2.3 Ce que nous NE collectons PAS

  • Localisation précise (GPS)
  • Carnet de contacts
  • Données de santé ou financières
  • Données biométriques
  • Cookies publicitaires tiers
  • Trackers Facebook / Google Ads

5. Tes droits (RGPD)

  • Accès : recevoir une copie de tes données → dpo@avis-base.com
  • Rectification : corriger une donnée fausse
  • Suppression : effacer ton compte directement depuis l'app (Profil → Paramètres → Supprimer mon compte)
  • Limitation : geler un traitement contesté
  • Portabilité : recevoir tes données en JSON
  • Opposition : refuser un traitement basé sur l'intérêt légitime
  • Réclamation : porter plainte à la CNIL (www.cnil.fr)

6. Suppression de compte — détail

Depuis l'app : Profil → Paramètres → Supprimer mon compte.

Sur clic, sous 30 jours :

  1. Ton compte Auth est supprimé immédiatement (impossibilité de se reconnecter)
  2. Tes articles publiés sont anonymisés (« Compte supprimé »)
  3. Tes commentaires sont anonymisés ou supprimés
  4. Tes messages privés sont supprimés
  5. Tes données techniques suivent leur cycle de rétention

Pour une suppression complète immédiate : dpo@avis-base.com.

7. Mineurs

L'app est interdite aux moins de 16 ans. Tout compte mineur détecté est supprimé.

8. Cookies

Le site n'utilise aucun cookie tiers publicitaire. Seul un cookie technique de session Supabase est utilisé pour te garder connecté.

9. Modifications

En cas de changement majeur, tu seras notifié par email + bandeau in-app.
